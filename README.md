# BurgerTime (Minigin Project)

This project is a recreation of the classic 1982 arcade game [**BurgerTime**](https://nl.wikipedia.org/wiki/BurgerTime), developed in **C++** using [**Minigin**](https://github.com/avadae/minigin).

It was created as part of the **Programming 4** course at **Digital Arts and Entertainment (DAE)**, where the goal was to recreate a classic 80’s arcade game while implementing several game programming patterns on top of the minigin engine.

---

## Features

- Recreation of the classic **BurgerTime**
- Built using **C++**
- Rendering and input handled through **SDL3**
- Vector math using **GLM**
- Custom engine built on top of **Minigin**
- Implementation of several **game programming patterns**
